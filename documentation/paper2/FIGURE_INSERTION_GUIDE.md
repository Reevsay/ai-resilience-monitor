## üé® Formatting Guidelines (Springer LNCS)

### Image Sizing
- **Full width figures:** Width = 12.2 cm (single column)
- **Recommended:** 85-90% of column width for margins
- **Charts/graphs:** Can be smaller (70-80%)
- **Tables:** Full width for readability

### Image Quality
- **Format:** PNG (already provided)
- **Resolution:** 300 DPI minimum (screenshots are high quality ‚úÖ)
- **File size:** All under 300KB ‚úÖ

### Caption Format
```
Figure X. [Caption text starts with capital, ends with period.]
```

Example:
```
Figure 1. System dashboard in idle state showing complete monitoring interface.
```

### Placement
- Place figure **after** first reference in text
- Use: "as shown in Figure X" or "see Figure X"
- Center align all figures
- Add 6pt spacing before and after

### Cross-Referencing in Word
1. Write: "The system architecture (Figure 1) shows..."
2. Insert ‚Üí Cross-reference ‚Üí Reference type: Figure
3. Select figure number
4. This auto-updates if figure numbers change
